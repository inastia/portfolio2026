Currency exchange app

Problem

Design an app and a responsive website to help global travelers find the current exchange rates between two currencies. They can also use this information to quickly and easily exchange currencies.

80% of the people I interviewed prefer to exchange currency at their destination.

Solution

I studied UX Design Course by Google on Coursera and built the app. I did created, planned, and conducted user interviews. I built empathy maps, a user persona, and a user flow.

I decided to create an app that allows to find the best rates for currency exchange at home as well as calculate the totals if the traveler wants to exchange currency on location.

User Research

I conducted supervised user research during low fidelity and hi fidelity phases of the design process. I did user interviews with 4 participants and based my user persona based on these four individuals. In this phase I found that most users exchange currency while at the destination location and only use Google to look up an approximate exchange rate before they leave to have an idea about their budget.

I created a user journey to better illustrate the difficulty many people face when they are budgeting for their trips abroad.

Empathy Map

One of the ways I tried to understand the user’s perspective was by creating an empathy map based on my interviews. I recorded and transcribed my interviews using Notes App on my computer. Then I manually added where I was speaking versus where the user was speaking. I then input the data into ChatGPT using a prompt to classify Says, Feels, Thinks, and Does for each user. ChatGPT was fairly accurate, but it made some errors. For example, it categorizes some of feelings in the Thinks section.

User Flow

Upon opening the app the user will be met with a calculator page. They will want to find the currency they have if it’s not the one already set be default. Then they will search for their destination currency and input the amount they are wanting to exchange. At this point they can review the mid-market rate or input their rate of the exchange kiosk they’re at.
