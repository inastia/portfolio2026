Problem

Design an app and a responsive website to help global travelers find the current exchange rates between two currencies. They can also use this information to quickly and easily exchange currencies.

80% of the people I interviewed prefer to exchange currency at their destination.

Solution

I studied UX Design Course by Google on Coursera and built the app. I did created, planned, and conducted user interviews. I built empathy maps, a user persona, and a user flow.

I decided to create an app that allows to find the best rates for currency exchange at home as well as calculate the totals if the traveler wants to exchange currency on location.
